/*binary tree*/

Each node in a tree has at most one parent, and one single node (called the root node), has no parent. A node that has no children is called a leaf of the tree. the root of any tree is an ancestor of every node in the tree. A subtree in a tree is any node in the tree together with all of its descendants. we can use trees to represent information that itself is hierarchical in nature: organizational charts and family trees. /// perfect - all leaves have same depth complete- always have 2 children on the left leaf
  • height(T) = 1 + max{height(TL), height(TR)}
  • if tree is empty then its height is -1; (n=0)
  • binray tree can have same nodes but different height
  • each node can have at most 2 children
  • root node is the first on top of tree
  • (n=1) - one tree (n=2) - two trees (n=3) - five trees
binary search tree - left leaf - lower child right leaf - higher child e.g root (8) first level- right(10) left (3) 2d level - root(3) left(1) right(6) 3rd level - root(6) left(4) right(7)
